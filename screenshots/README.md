This folder is reserved for screenshots of the application.

1. `admin-login.png` - The login screen.
2. `user-menu-desktop.png` - The main menu viewed on a desktop/laptop.
3. `user-menu-mobile.png` - The main menu viewed on a mobile device (you can use Chrome DevTools device toolbar).
4. `cart-overlay.png` - The menu with the shopping cart overlay open.
5. `admin-dashboard.png` - The admin panel showing the menu management table.
